# Personal Finance App

A simple personal finance application that displays a list of transactions and provides a summary of income, expenses, and balance.

## Tech Stack
- React
- TypeScript
- Tailwind CSS

## Features
- Display a list of transactions
- Show a financial summary:
  - Total income
  - Total expenses
  - Current balance
- Responsive layout for mobile and desktop screens

## Setup Instructions

```bash
cd assignment-2
npm install
npm run dev
```
---

## Design Decisions

- React and TypeScript are used to ensure type safety and maintainable code.

- Transaction calculation logic is separated into utility functions to keep components clean.

- Tailwind CSS is used for fast styling and mobile responsive design.
